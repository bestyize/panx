## 整体信息架构

- **顶栏：**
    - **左侧：** 产品 Logo、产品名（如「Panx 云盘」）、当前空间入口（个人 / 团队切换）。
    - **中间：** 全局搜索框（支持文件名、内容、标签、类型等），右侧附「高级筛选」图标。
    - **右侧：** 上传按钮（主按钮「上传」+ 下拉：上传文件 / 上传文件夹 / 新建文件夹）、通知中心、用户头像入口（账号、设置、退出）。

- **左侧导航：**
    - **主分区：** 首页、我的文件、与我分享、我分享的、最近、收藏、回收站。
    - **智能分区：** 图片、视频、文档、压缩包、音频、应用等「按类型」入口。
    - **运营/设置：** 会员/容量升级入口、设置、帮助与反馈。  [知乎专栏](https://zhuanlan.zhihu.com/p/1963639550686503541)  [UI Notes](https://uinotes.com/app/505837264627068013)

## 主文件区布局

### 1. 顶部工具条

- **路径与视图控制：**
    - **面包屑：** 根目录 / 工作 / 设计稿 / 2025-Q1（可点击跳转、右键在新标签打开）。
    - **视图切换：** 列表视图 / 网格视图切换按钮。
    - **排序与筛选：** 排序（时间 / 名称 / 大小 / 类型）、筛选（文件类型、时间范围、拥有者、标签）。  [腾讯云](https://cloud.tencent.com/developer/article/1794292)

- **操作按钮区：**
    - **主操作：** 上传、新建文件夹、新建文档（可接第三方在线文档）。
    - **批量操作：** 当有选中项时浮出「移动、下载、分享、重命名、删除、更多」操作条。

### 2. 文件列表视图

- **列结构：** 选择框、图标/缩略图、名称、标签、大小、修改时间、拥有者、操作。
- **交互细节：**
    - 悬停显示行尾「分享、下载、更多」图标。
    - 双击打开文件 / 进入文件夹。
    - 支持拖拽文件到左侧目录或标签上完成移动/打标签。
    - 支持按时间分组展示（本周、本月、更早），尤其在「分享」页。  [知乎专栏](https://zhuanlan.zhihu.com/p/1963639550686503541)

### 3. 网格视图

- **卡片内容：** 缩略图、文件名两行、省略号处理、右上角多选勾选框、右下角类型/时长（视频）。
- **适用场景：** 图片、视频、设计稿等视觉类内容；支持放大缩略图尺寸滑杆。

## 右侧信息/预览面板

- **触发方式：** 选中单个文件或文件夹时右侧滑出；再次点击关闭。
- **内容结构：**
    - **顶部：** 文件名、类型图标、快速操作（分享、下载、重命名）。
    - **预览区：** 图片/视频/文档在线预览；不支持的类型显示图标+「在本地打开」提示。
    - **属性区：** 大小、创建时间、修改时间、路径、拥有者、权限。
    - **协作区：** 评论、版本历史、最近活动（谁在何时做了什么）。  [CSDN文库](https://wenku.csdn.net/doc/6vywmibfrt)

## 搜索与智能能力

- **搜索入口：** 顶栏居中大搜索框，支持：
    - 关键词搜索（模糊匹配文件名、内容）。
    - 语义搜索（如「上周的合同 PDF」）。
    - 搜索建议：最近搜索、推荐筛选条件。  [优设网](https://www.uisdc.com/hunter_product/%e7%99%be%e5%ba%a6%e7%bd%91%e7%9b%98)

- **搜索结果页：**
    - 顶部保留搜索框+筛选条。
    - 左侧可按「类型 / 所有人 / 时间」快速过滤。
    - 结果支持列表/网格切换，支持按相关度排序。

- **AI 助手（可选增强）：**
    - 右下角悬浮按钮，点击打开对话框：
        - 「帮我找上次发给张三的合同」
        - 「清理大文件和重复文件」
    - 结果可直接跳转到对应文件或打开清理任务页。  [知乎专栏](https://zhuanlan.zhihu.com/p/1963639550686503541)  [优设网](https://www.uisdc.com/hunter_product/%e7%99%be%e5%ba%a6%e7%bd%91%e7%9b%98)

## 视觉与状态设计

- **整体风格：**
    - **色彩：** 以浅色背景+一到两个品牌主色（如蓝/紫）做强调；文件类型用不同色的小标签。
    - **层级：** 顶栏 > 左侧导航 > 主内容 > 右侧面板，使用阴影和分割线控制层次。
    - **图标：** 统一线性图标体系，文件类型图标颜色区分。

- **状态反馈：**
    - **空状态：** 提示文案 + 插画 + 引导按钮（如「上传文件」「新建文件夹」）。
    - **加载状态：** 骨架屏 + 进度条（上传/下载/同步）。
    - **错误状态：** 明确错误原因（网络、权限、空间不足）+ 推荐操作（重试、升级容量、联系拥有者）。

## 关键交互细节（体验拉满的地方）

- **拖拽上传：** 将文件拖入主区域，出现高亮边框和「松开上传到当前文件夹」提示。
- **多标签浏览：** 面包屑右键「在新标签中打开」，顶部以标签形式展示多个目录。
- **键盘操作：** 上下选择、Enter 打开、Delete 删除、Ctrl/Cmd+A 全选。
- **分享流程：**
    - 选中文件 → 顶部「分享」 → 弹出侧边/弹窗：
        - 链接分享（有效期、是否需要提取码、是否允许下载）。
        - 指定成员（搜索联系人、设置权限：可查看/可编辑）。
